###更改执行顺序  
您可以通过按住项目或按住屏幕左侧的“图标”并向上或向下移动来更改顺序。  
如果被选中，你不能改变顺序。请在重新排序前取消选择。  

### 从列表中删除  
点击复选框或任务列表名称以将其选中，然后点击屏幕底部的垃圾桶图标。  

＃＃＃ 添加到列表  
点击屏幕左下方的添加图标。
